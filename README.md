# HAOHE123.github.io

Lightweight identity and alias landing page for He Hao.

## Purpose

This site exists to capture name and handle searches such as:
- He Hao
- 郝赫
- Haohe
- HAOHE
- He HAO
- HAOHE123
- hehao123
- hehao111

The primary destination remains:
- https://haohe.me/

## Files

- `index.html` — single-page identity landing page
- `styles.css` — static styles inspired by the main resume page rhythm
- `robots.txt` — crawler entry rules
- `sitemap.xml` — sitemap for GitHub Pages indexing
- `site.webmanifest` — PWA metadata
- `assets/icons/` — favicon assets

## Deployment

Push the repo root to the `HAOHE123.github.io` repository and enable GitHub Pages from the default branch root.
